Sangu.be website
================

Building the website
--------------------

```bash
npm install
gulp
```

Testing the website
-------------------

```bash
npm run serve
```

and visit `http://127.0.0.1:8080/`

Configuration
-------------

Variables are stored in `./site/api/sp-config.json`  
Accessible as: `#{contributors}` in jade.

---------------------
